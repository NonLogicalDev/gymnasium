# ANTIGRAVITY.md - Gymnasium (Google Antigravity runtime)

Curated guidance for Google Antigravity agents.

This file is a working draft. Add rules here as real Antigravity sessions
surface them, then promote durable pieces into plugins under
`collection/plugins/`.

## Conventions

- Google Antigravity-specific plugins live under `collection/plugins/`.
- Each plugin directory should contain `plugin.json`.
- Validate plugins with `agy plugin validate <plugin-dir>` before publishing or
  installing them.
- Keep legacy Gemini CLI notes separate from current Antigravity plugin work.
